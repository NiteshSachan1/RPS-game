# RPS-game
This is the game project where a user a can play againt the computer.
There are three things to choose from, these are a rock, a paper and a scissors.
There are a total of three cases namely:
i) User Wins
ii) User Loses
iii) Match draws

And the respective conditions for the User to win are if User draws paper and computer draws rock or if the user draws rock and computer draws scissors or if user draws scissors and computer draws paper.
